# Python-Pong_Game

![Python_Pong_Game](https://i.postimg.cc/8c2x6j2z/GH6.png)

Pong is a 1972 sports video game developed and published by [Atari, Inc.](https://en.wikipedia.org/wiki/Atari,_Inc.) for arcades. It was created by [Allan Alcorn](https://en.wikipedia.org/wiki/Allan_Alcorn) as a training exercise assigned to him by Atari co-founder [Nolan Bushnell](https://en.wikipedia.org/wiki/Nolan_Bushnell). Pong was the first commercially successful video game,[4] helping to establish the video game industry along with the Magnavox Odyssey.

I have implemented a simplified version of the game using python turtle graphics.
